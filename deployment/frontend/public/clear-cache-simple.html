🔧 캐시 클리어 가이드

⚠️ 현재 문제: 브라우저가 오래된 캐시를 사용해서 onrender.com으로 요청을 보내고 있습니다.
1 Chrome 브라우저에서 사이트 열기

아래 링크를 클릭하여 사이트를 여세요:

🌐 사이트 열기 (새 탭)
2 개발자 도구에서 캐시 삭제

사이트가 열린 후:

  1. F12 키를 눌러 개발자 도구 열기
  2. Console 탭 클릭
  3. 아래 코드를 복사해서 붙여넣고 Enter 누르기:
// 모든 캐시 삭제 및 새로고침 localStorage.clear(); sessionStorage.clear(); console.log('✅ 캐시 삭제 완료!'); setTimeout(() => location.reload(true), 1000);
✅ 코드가 클립보드에 복사되었습니다!
3 강제 새로고침 (추가 확실)

캐시 삭제 후에도 문제가 있다면:

  • Ctrl + Shift + R 누르기
  • 또는 새로고침 버튼을 우클릭 → "캐시 비우기 및 강력 새로고침"

🚨 최후의 방법: Chrome 캐시 완전 삭제

Chrome 설정에서 모든 캐시 삭제:

  1. Chrome 설정 열기 (⋮ → 설정)
  2. "개인정보 및 보안" → "인터넷 사용 기록 삭제"
  3. "캐시된 이미지 및 파일" 체크
  4. 시간 범위: "전체"
  5. "인터넷 사용 기록 삭제" 클릭
